# Shopping

Simple shopping app 
## Stack
- React 17
- Typescript 4
- TailwindCss

> This is purely a frontend repository.
## Setup

```bash
# install dependencies
$ npm install
```

## Development

```bash
# start development
$ npm run start
```

## Testing

```bash
# run test
$ npm run test
# watch
$ npm run test:watch
# update snapshots
$ npm run test:update
